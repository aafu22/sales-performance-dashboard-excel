# Sales Performance Dashboard (Excel)

This project analyzes sales data using Microsoft Excel and presents insights through pivot tables and dashboards.

## Features
- Raw sales data cleaning
- Pivot table analysis
- Sales by region and product
- Monthly sales trend
- Interactive dashboard

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Basic Excel formulas

## Project Highlights
- Region-wise sales analysis
- Product-wise performance
- Monthly trend visualization
